# Course Alignment — mini-feedforward-control

## Nine-School Curriculum Mapping

| School | Course | Feedforward Topics Covered |
|--------|--------|---------------------------|
| **MIT** | 6.302 Feedback Systems | Feedforward compensation, 2-DOF architecture |
| **MIT** | 6.243 Nonlinear Control | Computed torque FF, feedback linearization |
| **Stanford** | ENGR105 Feedback Control | 2-DOF control, prefilter design |
| **Stanford** | ENGR209A Nonlinear Control | Nonlinear FF, adaptive FF |
| **Berkeley** | ME232 Advanced Control | Motion control FF, input shaping |
| **Berkeley** | ME234 Nonlinear Control | Computed torque for robotics |
| **Caltech** | CDS 212 Robust Control | Robust 2-DOF, small-gain analysis |
| **ETH** | 151-0591 Control I | Storungsgrossenaufschaltung (disturbance FF) |
| **ETH** | 151-0567 MPC | ILC, learning feedforward |
| **Cambridge** | 3F2 Systems & Control | Disturbance feedforward, 2-DOF |
| **Cambridge** | 4F2 Robust Control | Robust feedforward design |
| **Georgia Tech** | AE 6530 Optimal Control | Input shaping for spacecraft |
| **Purdue** | ME 675 Multivariable | 2-DOF multivariable FF |
| **Tsinghua** | Automatic Control Theory | Qian kui kong zhi (feedforward control) |
| **Tsinghua** | Modern Control Theory | State-space FF, observers |
